The PC Navigator Page:
Browse Your Computer Files Now!

 

Most browsers will disable most of the features of this webpage because it is on the Internet.

This webpage will work better and have more functionality on most computers if it is first saved as an HTML file on your computer.

Then, because it is a local file and not a webpage on the Internet, more of the features will become functional that way.


 

If you have already saved this file on your computer, then the content on this webpage should allow you to browse and open local hard drives, local disks and files saved on your computer.  Some users may need to perform a couple of extra steps.  See if a yellow bar appears above, saying something like, "(Your browser's name here) has restricted this file from showing active content that could access your computer."  If the yellow bar is present, click on it and "Allow blocked content..." (or a choice like that with similar wording) then click the "Yes" button.  The contents of this webpage should now be completely operational, unless your computer is running security software that prohibits you from completing these instructions, granting permission to allow blocked content or from viewing certain files or drives.

For some users using some browsers, the above instructions may need to be repeated every time you open or navigate back to this page.

This page has several parts that may not work on many computers if you are viewing this directly from the Internet.  This security measure prevents websites from trying to access your computer files without your knowledge or consent.  However, this webpage will work on most computers if it is first saved as an HTML file on your computer.

Nevertheless, there are still a few ways how one can browse and view hard drives and files directly from within most browsers.

To begin, try clicking on the links below.  Some should work, but most of them will not do anything.  Click on the links with the letter "c" first.  If nothing happens, highlight the link text, then Copy and Paste it into the address bar of your browser, then click Go or press Enter, and the hard drive contents should be displayed right here in your browser.  Whether any of this works or not will depend upon the security measures and computer settings on your computer and local network.  For example, if you delete everything in the address bar, and just type the letter "c", a colon ( : ) and a backslash ( \ ), then click Go or press Enter, the hard disk contents of the C drive should be displayed right here in your browser, unless a security setting prohibits you from viewing the C drive.

 

The Open or Upload buttons on this page will not open or upload anything.
These buttons are just a way to make your browser let you look around at the local folders and files on your computer.
These buttons might also help your browser show you the folders and files in your Local Area Network (LAN) or local shared drive or locally shared hard drives.



File System Protocols:

The links below, by themselves, will not work.
You need to know the exact name of the exact directory, and add it after the second forward-slash.

For Macintosh machines, try this first...
Service Message Block (SMB)
smb://

For Windows machines, try this first.
Some web browsers know how to navigate File System Protocols, sometimes, to some places.
The File Uniform Resource Identifier (URI) is most commonly used in Windows.
file://

Apple File Protocol (AFP)
afp://


Network File System (NFS)
nfs://


Common Internet File System (CIFS)
cifs://

AFP is going to support Apple's resource forks and attributes natively.
For Macintosh machines,
SMB/CIFS will by default dump these attributes into a new folder
.DS_Store is a file placed into all directories that a Macintosh machine navigates.
If a Windows machine shows a .DS_Store folder in a directory,
it is because a Macintosh machine had connected to and navigated in that directory.

The two items below should also work for Macintosh Netscape users. A Pathway looks and works very much like a URL or Internet address, except a pathway is used to open a file on your computer, instead of a webpage on the Internet,
and a pathway uses backslashes ( \ ) instead of the traditional forward-slashes ( / ).

To open a specific file directly, you will need to know the pathway or address that leads to the exact location of that file.  The following two items will help you at least determine what the pathway is.  You can browse the local drives and disks using the items below.  Click on "Browse" to begin browsing in a small, pop-up window.  This should work across the Internet on most computers, and it should work for Macintosh users as well.  When you find the file of interest, click on the Open button in the pop-up window.  The file will not be opened.  Instead, the pop-up window will close and the pathway will be displayed in the white box.  Click Open to see if the file will open-up.  If nothing happens, highlight all of the pathway text (click in box and Select All or Ctrl+A), then Copy (Ctrl+C) and Paste (Ctrl+V) it into the address bar of your browser, then click Go or press Enter, and the file should be displayed right here in your browser.  This should work because you – not the webpage – are personally giving the command by typing or entering it into the address bar yourself.  This may not work if a security setting prohibits you from viewing that file or drive.
